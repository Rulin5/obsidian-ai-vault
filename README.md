# Obsidian AI Vault

一个基于 Obsidian + AI 的个人知识库、项目管理与复盘模板。

它适合用来管理：

- 个人项目
- 月度规划
- 周计划
- 每日复盘
- 反思分类
- 想法收集
- 长期领域沉淀
- 决策记录

## 核心流程

```text
长期方向
-> 项目目标
-> 月度核心结果
-> 每周交付成果
-> 每日具体任务
-> 实际完成情况
-> 偏差分析
-> 后续调整
```

## 反思分类流程

```text
原始反思
-> 原文保留
-> 原子反思拆分
-> 分类标签
-> 归属判断
-> 原则提炼
-> 待规划任务
-> 月/周计划时再决定是否执行
```

## 使用方式

1. Clone 或下载这个仓库。
2. 用 Obsidian 打开仓库文件夹。
3. 打开 `个人项目系统/README.md`。
4. 把 `个人项目系统/00-控制中心/当前状态.md` 改成自己的真实状态。
5. 在 AI 插件中先让 AI 读取 `AI总规则.md` 和 `当前状态.md`。

推荐开场提示：

```text
请先读取 个人项目系统/00-控制中心/AI总规则.md 和 个人项目系统/00-控制中心/当前状态.md，然后告诉我这个系统应该如何开始使用。不要修改文件。
```

## 设计原则

- 先预览，再写入。
- 反思不直接变任务。
- 待规划任务不直接进入月计划或周计划。
- 未完成任务不自动顺延，必须重新判断价值和优先级。
- 每月最多 3 个核心结果。
- 每周最多 2 个核心交付成果。
- 每天最多 3 个主要任务。
