# 02-长期领域

长期领域用于沉淀没有明确结束时间、但需要持续积累的能力和主题。

## 和项目的区别

- 项目：有明确结果和截止时间。
- 长期领域：持续积累原则、方法、经验和判断标准。

## 反思归属规则

反思先进入 [[个人项目系统/05-想法收集箱/反思收集箱]]，由 AI 分类后再沉淀到对应长期领域。
