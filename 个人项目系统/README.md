## 使用流程

### 月初

```text
执行月度规划
```

### 周初

```text
执行本周计划
```

### 每天

```text
执行今日复盘
```

### 周末

```text
执行本周复盘
```

### 反思分类

```text
执行反思分类
```

反思会先进入分类预览，可执行内容只会进入 [[个人项目系统/05-想法收集箱/待规划任务|待规划任务]]，不会直接打乱月计划或周计划。

## 写入规则

AI 需要先生成预览。确认无误后，再发送：

```text
确认写入
```
