# 使用说明

## 第一次使用

1. 用 Obsidian 打开当前文件夹。
2. 打开 `个人项目系统/00-控制中心/当前状态.md`。
3. 填写当前月份、当前周、当前重点项目和主要风险。
4. 打开 AI 插件，发送：

```text
请先读取 个人项目系统/00-控制中心/AI总规则.md 和 个人项目系统/00-控制中心/当前状态.md，然后告诉我这个系统应该如何开始使用。不要修改文件。
```

## 月初

```text
执行月度规划
```

AI 会读取当前状态、项目、长期领域、想法收集箱和待规划任务，生成月度规划预览。

## 周初

```text
执行本周计划
```

AI 会根据月度总控、上周复盘和待规划任务生成本周执行计划。

## 每天

```text
执行今日复盘
```

AI 会先提问，再生成今日复盘预览。

## 周末

```text
执行本周复盘
```

AI 会统计本周目标、实际结果、偏差原因和下周建议。

## 反思分类

```text
执行反思分类：
粘贴你的反思
```

AI 会把一段混合反思拆分成原子反思，分类并提炼原则。可执行内容只进入待规划任务，不会直接打乱计划。

## 写入规则

AI 修改文件前必须先生成预览。

确认写入：

```text
确认写入
```

要求修改预览：

```text
修改：具体修改要求
```

取消：

```text
取消写入
```
